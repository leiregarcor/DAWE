1# CONEXIÓN A INSTANCIA EN DIGITAL OCEAN

1- abrir digital ocean:

https://cloud.digitalocean.com/projects

2- copiamos la ip: 178.128.162.177, después abrimos una terminal y nos conectamos a la instancia:

ssh root@178.128.162.177




2# SERVIDOR LOCAL NODE

1- crear una carpeta de trabajo:

mkdir dawe001
cd dawe001

2- para crear un servidor:

npm init

3- pulsamos enter en todo lo que pida, al final => 
	Is this OK? (yes) yes -> genera un package.json

4- instalamos el paquete 'serve'

npm install serve

5- se crean los archivos:

ls -l

	node_modules/
	package-lock.json
	package.json

6- comprobamos las dependencias instaladas en package.json

cat package.json
	{
	  "name": "dawe005",
	  "version": "1.0.0",
	  "description": "",
	  "main": "index.js",
	  "scripts": {
	    "test": "echo \"Error: no test specified\" && exit 1"

	  },
	  "author": "",
	  "license": "ISC",
	  "dependencies": {
	    "nerve": "^1.0.0"
	  }
	}


7- creamos una carpeta para los ficheros dentro de la carpeta de trabajo:
mkdir puclic

8- para servir los ficheros dentro de esa carpeta (public), modificaremos el package.json:

gedit package.json

	{
	  "name": "dawe005",
	  "version": "1.0.0",
	  "description": "",
	  "main": "index.js",
	  "scripts": {
	    "start": "serve ./public"
	  },
	  "author": "",
	  "license": "ISC",
	  "dependencies": {
	    "nerve": "^1.0.0"
	  }
	}
	
	
	
	
3# SUBIR EL EJERCICIO A DIGITAL OCEAN

1- abrimos una terminal en la ruta donde están los archivos que queremos subir

2- podemos subir un fichero usando el comando “scp”:

scp <nuestro fichero> root@178.128.162.177:/root/dawe005/public

2- bien subir varios archivos:

scp -r <nuestro ficheros> root@178.128.162.177:/root/dawe005/public
